Title: Google Pay Expense Sharing System(GPESS)

Technology Stack:

   Programming language used Python
   Libraries used:
   1) Numpy
   2) PrettyTable

Core Features:

  Add, split, and track expenses among multiple users.
  Implements equal and custom split functionalities.
  Maintains a transaction history for each user.

Functional Features:

   Allows users to add expenses with descriptions and amounts.
   Automatically calculates and displays shared amounts.
   Provides a summary of balances between users.

Available both as Jupyter Notebook and as well as a Python Source File
